# Python-csv

Example python read csv file with | delimeter

## Prerequisite

- python
- pip

## Steps

```sh
pip install virtualenv
python3 -m venv venv
source venv/bin/activate
```

RUN

```python
python main.py
```

Put dependency to file

```sh
pip freeze -r  requirements.txt > requirements.txt
```

Deactivated

```sh
deactivate
```
